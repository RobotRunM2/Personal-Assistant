# 🙋‍♀️ Personal-Assistant

万能通用的，签到，打卡，查询，提醒，通知，等一切基于 http(s)协议的操作，你的私人助理，帮你完成一些无聊的任务。

## 🍳 具体能干点啥，随便来点例子

- 什么值得买 签到打卡
- 联通 APP 签到领流量
- PT 站签到保号
- HOSTLOC 签到上元老
- V2EX 签到领金币
- 重要日期提醒
- 火车票余量监控
- 天翼云盘每日轮盘领容量
- 每日健康码打卡
